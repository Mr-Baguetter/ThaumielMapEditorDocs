# 🔷 ChamberData

<Badge color="blue">class</Badge> <Badge color="secondary">`ThaumielMapEditor.API.Blocks.ServerObjects.Lockers`</Badge>

Describes a single item entry that can be spawned inside a locker chamber.

## Overview

### Properties

| Name | Type | Access | Description |
|------|------|:------:|-------------|
| [`AmountToSpawn`](#amounttospawn) | `int` | `get` `set` | Gets or sets the Number of instances to spawn when this entry is selected. |
| [`ItemType`](#itemtype) | `ItemType` | `get` `set` | Gets or sets the type of item to spawn. |
| [`SpawnPercent`](#spawnpercent) | `float` | `get` `set` | Gets or sets the chance this entry is selected when populating the chamber, expressed as a percentage (e.g., 50.0 = 50%). |
